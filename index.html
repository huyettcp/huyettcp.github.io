Sabergraphs

Sabergraphs is a data visualization of MLB team's payrolls and player's cost vs. performance score. This was my final project for General Assembly's Web Development Immersive.

Technologies Used: Ruby on Rails, rSpec, Javascript, jQuery, D3.js, Nokogiri, Ajax

Flosswords

Flosswords is an infinite photo crossword puzzle that contains words from New York Times headlines and picture clues from Flickr.

Technologies Used: Ruby on Rails, PostgreSQL, CWORD JavaScript Crossword Engine, Flickr API, Nokogiri, Ajax

Box Office Visualization

This is a D3.js visualization of box office data. It features the highest grossing movie from each month from 2010 to 2013.

Technologies Used: Ruby on Rails, rSpec, Javascript, jQuery, D3.js, Nokogiri, Ajax, OMDB api
